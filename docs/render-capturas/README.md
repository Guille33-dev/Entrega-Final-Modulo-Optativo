# Capturas para completar la entrega

Cuando despliegues el proyecto en Render, añade aquí las capturas solicitadas por el enunciado:

1. `01-health-render.png`: ruta `/health` funcionando en Render.
2. `02-render-health-config.png`: configuración de Render con `Health Check Path` apuntando a `/health`.
3. `03-render-logs.png`: logs visibles en Render al hacer peticiones a la API.
4. `04-github-actions.png`: workflow de GitHub Actions ejecutado correctamente.

Estas capturas no se pueden generar antes del despliegue real porque dependen de tu cuenta de GitHub y Render.
